# Suisse Intl Fonts

Download these fonts from [OnlineWebFonts - SuisseIntl](https://www.onlinewebfonts.com/fonts/SuisseIntl) and place the `.woff2` files here:

- `SuisseIntl-Regular.woff2` (weight 400)
- `SuisseIntl-Medium.woff2` (weight 500)
- `SuisseIntl-SemiBold.woff2` (weight 600)
- `SuisseIntl-Bold.woff2` (weight 700)

Select **woff2** format when downloading. The app will build and run without these files (using system fallback); add them to enable Suisse Intl.
