# LITHE · lithe.pw

Personal portfolio website inspired by [byappi.com](https://byappi.com/). Built with Next.js and Tailwind, deployed on Vercel.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

Edit `src/app/page.tsx` to update:

- **CONFIG** – site name, tagline, CTA link, location, nav
- **STACK** – your services and tech
- **PROCESS** – your workflow steps
- **PORTFOLIO** – your projects

## Deploy on Vercel

1. Push to GitHub
2. Import the repo at [vercel.com](https://vercel.com)
3. Add your domain `lithe.pw` in project settings

Vercel will detect Next.js and configure the build automatically.
